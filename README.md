Chitter
=================

A twitter clone where you can see a public stream of 'Peeps' posted by users and sign up to post Peeps yourself.

Visit https://louisa-chitter.herokuapp.com/ to play with the app.

Technologies used:
-------
* Bcrypt
* Datamapper
* Postgres
* Sinatra
